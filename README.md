# Atlassian Rovo 實戰課程系列

Atlassian Rovo AI 課程網站，提供從入門到高階的完整課程系列。

## 課程內容

| 課程 | 難度 | 時長 | 說明 |
|------|------|------|------|
| Rovo AI 免費入門 | 入門 | 60 分鐘 | Search、Chat、Agent 三大核心功能 |
| Rovo Agent 開發實戰 | 初階 | 90 分鐘 | 從零建立 Agent，學會 Instructions 與 Actions |
| Rovo Agent + Automation | 中階 | 120 分鐘 | JSON 輸出與批次建立 User Stories |
| Rovo Multi-Agent 工作流程 | 高階 | 180 分鐘 | 多 Agent 協作與自動化工作流程 |

## 專案結構

```
atlassian-rovo-course/
├── index.html              # 主要 HTML 頁面
├── css/
│   └── styles.css          # 所有樣式定義
├── js/
│   └── main.js             # JavaScript 互動功能
├── assets/
│   └── icons/              # SVG 圖示資源
└── README.md
```

## 技術說明

- **HTML5** - 語意化標籤結構
- **CSS3** - CSS Variables、Flexbox、Grid、響應式設計
- **Vanilla JavaScript** - 無框架依賴，使用 Intersection Observer API
- **Google Fonts** - Noto Sans TC 中文字型

## 本地開發

直接在瀏覽器中開啟 `index.html` 即可預覽：

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

或使用本地伺服器：

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve
```
